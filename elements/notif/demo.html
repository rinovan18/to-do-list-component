Notification Element for HAXCMS

Basic Usage

This is a vanilla JavaScript Lit Element designed for HAXCMS integration.

HAXCMS Integration

To use this element in HAXCMS, add it to your site's assets and include it in your pages:

<!-- In your HAXCMS page HTML -->
<script type="module" src="/assets/@notif/notif-element.js"></script>

<!-- Use the element -->
<notif-element
  type="info"
  message="Welcome to HAXCMS!"
  duration="5000"
  closable="true"
></notif-element>
